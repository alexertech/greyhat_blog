# Greyhat Blog Engine

A blog engine written entirely in Ruby on Rails + PostgreSQL with [Primivite UI](https://taniarascia.github.io/primitive/) as CSS template. The live version is available at: [greyhat.cl](http://www.greyhat.cl)
